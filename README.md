# PR09 - Apps Script API + Android (Jetpack Compose)

App Android en Jetpack Compose (MVVM + Retrofit + LiveData) que consume una API REST hecha con Apps Script para **leer** datos desde Google Sheets y **insertar** nuevos registros con POST. Incluye login/registro y guarda la sesión en `SharedPreferences`.
